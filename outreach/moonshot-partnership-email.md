## What is Kimi Enterprise?

Kimi Enterprise simulates a complete software engineering organization using AI agents:

```
CEO → CTO → VPs → Directors → Engineering Managers → ICs
```

**Key Features**:
- 🏗️ **Hierarchical Organization**: 12+ agent roles from CEO to Junior Dev
- 💰 **Cost Control**: Real-time budget tracking with circuit breakers
- 📈 **Auto-Scaling**: Dynamic agent scaling based on workload
- 🔗 **Git Integration**: Automatic PRs, code reviews, conventional commits
- 📊 **Rich Dashboard**: Monitor your AI team in real-time

**GitHub**: https://github.com/abdesselam-tech/kimi-enterprise  
**Stars**: Growing community

## Technical Integration Details

### MCP Server Implementation

Kimi Enterprise exposes an MCP server that Kimi CLI can communicate with:

```json
{
  "mcpServers": {
    "kimi-enterprise": {
      "command": "python",
      "args": ["-m", "kimi_enterprise.mcp.bus_server"],
      "env": {
        "KE_PROJECT": "/path/to/project"
      }
    }
  }
}
```

### API Compatibility

- Uses standard MCP protocol
- No proprietary extensions
- Works with existing Kimi CLI tooling
- Easy to extend
